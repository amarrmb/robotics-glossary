## Why This Glossary?

The robotics documentation landscape is fragmented. NVIDIA PDFs, ROS wikis, academic papers, Discord threads — knowledge is scattered across dozens of sources.

This glossary brings it together in one **hyperlinked knowledge graph** organized by domain, with content tagged by type:

  <span class="level-badge practical">Practical</span> Hands-on guides
  <span class="level-badge deepdive">Deep Dive</span> Comprehensive coverage
</div>
